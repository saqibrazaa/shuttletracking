# Shuttle Tracking System

This project is a full-stack Shuttle Tracking System with:
- React.js frontend (student, driver, admin dashboards)
- MongoDB Atlas backend (Atlas App Services/Stitch)
- Real-time location tracking
- Google Maps API integration
- Push notifications (Firebase Cloud Messaging)

## Project Structure

- `frontend/` — React.js web app
- `backend/` — Stitch/Atlas App Services functions, schema, and integration notes
- `infrastructure/` — Deployment, environment, and configuration files

## Quick Start

1. **Frontend**
   - Navigate to `frontend/`
   - Run `npm install` and `npm start`

2. **Backend**
   - Set up MongoDB Atlas cluster
   - Configure Atlas App Services (Stitch) with collections and functions (see `backend/`)

3. **Maps & Notifications**
   - Add Google Maps API key in frontend
   - Set up Firebase Cloud Messaging for push notifications

4. **Deployment**
   - Frontend: Deploy to Vercel/Netlify
   - Backend: Atlas App Services (cloud hosted)
