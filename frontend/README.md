# Shuttle Tracking System Frontend

This folder will contain the React.js web application for students, drivers, and admins.

## Setup
1. Run `npx create-react-app .` (or use Vite/Next.js if preferred)
2. Add pages/components for:
   - Login/Signup
   - Student Dashboard (Map, Shuttle Tracking, Routes, Notifications)
   - Driver Dashboard (Trip Start, Live Tracking)
   - Admin Dashboard (Overview, Shuttle/Route/User Management, Live Monitoring)
3. Integrate Google Maps API
4. Connect to backend via Stitch/Realm Web SDK
5. Add FCM for push notifications
