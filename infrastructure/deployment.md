# Infrastructure & Deployment

## Frontend
- Deploy to Vercel or Netlify
- Set environment variables for API endpoints and Google Maps key

## Backend
- Atlas App Services (Stitch) is cloud hosted
- Configure rules and triggers in Atlas UI

## Database
- MongoDB Atlas (cloud hosted)
- Set up collections as per backend/schema.md

## Notifications
- Use Firebase Cloud Messaging (FCM)
- Add FCM config to frontend for push notifications
